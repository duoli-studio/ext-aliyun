```
$ php artisan ide-helper:model "System\Models\PamAccount"
```

## 事件

1. 事件命名是 DoWhatEvent
2. 数据的修饰符为 public ,无需定义 get 方法
3. 事件需要继承 Poppy\Framework\Application\Event
4. 事件监听器必须为 DoWhatSendImListener, 事件中需要体现 监听器的作用
